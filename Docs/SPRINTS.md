## 💡 LESSONS LEARNED

| # | Lesson |
|---|---|
| 1 | Always close HTML tags properly — missing closing tags break entire layout |
| 2 | Test on mobile early — don't wait till Sprint 4 |
| 3 | Use clamp() from the start for fluid typography |
| 4 | CSS position:absolute + inset:0 is the key trick for image-in-card |
| 5 | IntersectionObserver is better than scroll events for animations |
| 6 | Compress images before adding — large files slow page significantly |
| 7 | CSS variables save enormous time when changing colors or spacing |
