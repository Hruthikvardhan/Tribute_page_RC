## 🎯 About the Project

A fully responsive, modern tribute page dedicated to **Ram Charan** — Telugu Superstar, Actor, Entrepreneur and Global Icon. This project was built as part of my frontend development learning journey to practice real-world HTML, CSS, and JavaScript skills without using any frameworks.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎨 Dark Theme | Cinematic dark theme with red and gold color scheme |
| 📱 Fully Responsive | Works perfectly on mobile, tablet, and desktop |
| 🍔 Hamburger Menu | Full-screen mobile navigation overlay |
| ✨ Scroll Reveal | Elements animate in as you scroll |
| 🌟 Particles | Floating gold particle effects in hero |
| 🔢 Counter Animation | Stats count up on page load |
| 🏆 Timeline | Zigzag filmography timeline |
| 🎭 Smooth Scroll | Smooth scroll navigation between sections |
| 🖼️ Photo Integration | Hero background + portrait photo |
| 🔖 Favicon | Custom RC favicon |

---

## 🛠️ Tech Stack

```
Frontend Only — No Frameworks Used

├── HTML5          → Semantic markup, ARIA accessibility
├── CSS3           → Variables, Grid, Flexbox, Animations
├── JavaScript     → ES6+, DOM manipulation, IntersectionObserver
└── Google Fonts   → Cinzel Decorative, Cinzel, Lato
```

## 🚀 Getting Started

### View Locally

```bash
# Step 1 — Clone the repository
git clone https://github.com/Hruthikvardhan/Tribute_page_RC.git

# Step 2 — Navigate into folder
cd ram-charan-tribute

# Step 3 — Open in browser
# Just double click index.html
# OR right click → Open with → Chrome
```

No npm install needed. No build step. No dependencies. Just open and run! ✅

## ☁️ Deployment

Deployed on **Vercel** via GitHub integration.

Every `git push` to `main` branch auto-deploys to production.

```bash
git add .
git commit -m "your message"
git push origin main
# → Vercel auto deploys in ~30 seconds ✅
```
